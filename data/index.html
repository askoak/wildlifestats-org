Data

National database

The national database is the query interface to the dataset. It will offer free-text search and dropdown filters across species, taxonomic class, intake reason, geography, and season, with results presented as tables and charts. Downloads are available as CSV, subject to a suppression rule that withholds any cell below a fixed count so that no figure can be resolved to an individual case. The interface is wired to the synthetic dataset now and will track real data as it is contributed under the tiered access model.

This section is under construction. The query interface is wired to the dataset in Phase 3 of the build plan.

WildlifeStats is a research framework. Current dataset is synthetic (n=100,000) generated from regional distribution models. See Methodology.
