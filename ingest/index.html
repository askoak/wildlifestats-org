Ingestion sandbox

Multi-format ingestion sandbox

Wildlife rehabilitation records live in thousands of incompatible spreadsheets, each with its own columns, abbreviations, and conventions. The ingestion sandbox documents how WildlifeStats reconciles them: how a heterogeneous set of files is read, cleaned, mapped to a common schema, and merged into the national database while preserving each contributor's provenance. This page is a methodology demonstration rather than a live upload tool, illustrated with sample files that show the standardization workflow end to end. It explains how the framework would scale to many contributing centers without requiring real partner data to be present.

This section is under construction. The demonstration lands as part of Phase 4 of the build plan.

WildlifeStats is a research framework. Current dataset is synthetic (n=100,000) generated from regional distribution models. See Methodology.
